OUTPUT FORMAT REQUIREMENTS:

हर स्लाइड के लिए:
- पूरा React component code
- motion.div / AnimatePresence का सही उपयोग
- Clean, production-ready formatting
- कोई TODO या placeholder नहीं
- Logical component naming
- **PascalCase naming:** सभी React components और filenames PascalCase में हों  
  (उदा. `SlideTitleCard.jsx`)
- **Valid Identifiers:** Component names और imports valid JS identifiers हों  
  **Dots (.) allowed नहीं हैं**
- **No Messy Filenames:** double underscores या double extensions नहीं
- **Single Extension:** सिर्फ एक `.jsx` extension
- **Setup_Instructions.jsx generate नहीं करना**  
  (सभी dependencies पहले से installed हैं)

Filename उदाहरण:
Slide1_Title.jsx  
Slide2_Concept.jsx  
Slide3_Flow.jsx  

**Bundling:**
पूरा code एक ही Markdown response में दें।
हर फ़ाइल को Level 2 Header से अलग करें:
`## Filename.jsx`
उसके नीचे code block दें।
